Street View API Wrapper Readme File for offline Asset Package assistance.

Description:
Street View API Wrapper helps in delivering Street View content into your project

Instructions:
To use Texture Noise Compression, please follow these steps.

1.  Add a Cubemap into your scene (Sphere, or Skybox)

2.  Add the StreetViewCube component

2.  Enter your Google Street View API Key into the component (You can obtain an API key here: https://developers.google.com/maps/documentation/streetview/ )

3.  Specify the Latitude Longitude, and link in your cubemap to the component

4.  That's it!   Run the program, and it'll make the API request, and load the Street View into your cubemap
